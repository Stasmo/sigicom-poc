# Datalink App

Syncs data from sigicom to vdv.

## Installation

```
npm i
```

Then add your secrets to the .env.example file.

## Running the app

```
npm start
```

The application will run on port 8080 by default. To run it on
a different port.

```
PORT=8088 npm start
```
